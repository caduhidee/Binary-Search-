# Binary Search 

This site demonstrates binary search, where you can build your own binary trees and search for inputted values.

## How to run
Step 1: After downloading all files from GitHub, open the project in an IDE.

Step 2: In the terminal, run:
```console
cd backend
mvn clean install
mvn spring-boot:run
```
to initialize the backend.

Step 3: In a new terminal, run:
```console
cd frontend
npm start
```
to initalize the frontend.

Step 4: Go to [http://localhost:3000/](http://localhost:3000/) and enjoy!
